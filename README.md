# Vignette 3D Portfolio

A premium, ultra-modern 3D portfolio website for "Vignette" featuring a floating 3D hero section, glassmorphism design, and responsive video grids.

## Features
- **3D Hero**: Interactive Torus Knot using Three.js.
- **Responsive Layout**: Works on Mobile, Tablet, and Desktop.
- **Video Grid**: Highlights section featuring autoplaying, muted videos.
- **Services**: 2x2 Grid optimized for mobile.

## Technology
- HTML5 / CSS3 / JavaScript (Vanilla)
- TailwindCA (via CDN)
- Three.js (via CDN)
- GSAP (via CDN)

## How to Run
Just open `index.html` in your browser. No build step required.
